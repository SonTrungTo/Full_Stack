Some basic syntax:

Basic functionals

print(args): Print the args out on the output console. E.g, print("I am legend!") should yield I am legend! on the console.

operator(arg1, arg2): operator = [+, -, *, /, %, ==, <, >]. E.g, +(1, 1) should yield 2.

array(args): enable the array functionality for program. E.g, array(1,2,3) is equivalent to [1, 2, 3].

length(array) returns the length of an array, the equivalence of array.length in JS.

element(array, n) returns the nth element of the array.

Basic conditional statements and function calls

if(arg1, arg2, arg3) is equivalent to if(arg1) arg2; else arg3;

while(arg1, arg2) is equivalent to while(arg1) arg2;

do(args) is equivalent to {arg1; arg2; ...}

define(arg1, arg2) is equivalent to let arg1 = arg2;, binding arg2 to arg1.
